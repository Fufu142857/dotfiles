# fufu's dotfiles 🌙

Minimalist configuration for macOS. 

> "Code is poetry, terminal is home."

## 🛠 Tech Stack
- **Editor:** Neovim (`init.vim` based)
- **Shell:** Zsh

## 📂 Project Structure
## 🔧 Highlights
- **`trans`**: Unified translation utility with dynamic ANSI coloring.
- **`vscode_theme.vim`**: A customized Neovim theme migrated from my VSCode era.

## 🚀 Sync
Update configurations by running `dotsync` in any terminal session.
